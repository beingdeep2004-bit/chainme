## How to Use

1. **Visit** https://chainme.vercel.app
2. **Type** any name or text (use A-Z letters)
3. **Generate** — Click Generate or press Enter
4. **Watch** — Black cards flip to reveal crypto symbols
5. **Download** — Get a professional symbol-only image
6. **Share** — Tweet it or copy the link

## Design System

**Color Scheme:**
- Background: Linear gradient from `#0a0a0a` → `#1a1a1a` → `#0f0f0f`
- Panel: Transparent glass with `rgba(30, 30, 30, 0.6)` + 50px backdrop blur
- Borders: Subtle `rgba(255, 255, 255, 0.08)` + inset glow
- Text: Clean whites and grays on dark background

**Typography:**
- Font: System San Francisco (-apple-system, BlinkMacSystemFont)
- Spacing: Generous padding and letter-spacing for clarity
- Shadows: Deep, subtle depth with inset highlights

**Cards:**
- Placeholder: Solid black with subtle shimmer
- Revealed: Transparent glass with symbol only
- Animation: 600ms smooth 3D flip
- Stagger: 150ms between each card

## Technical Stack

- **HTML5** — Semantic structure
- **CSS3** — Glass morphism, 3D transforms, backdrop blur
- **Vanilla JavaScript** — No frameworks, no dependencies
- **Canvas API** — Image generation and download
- **Web Audio API** — Procedural tick sounds
- **Symbols & Unicode** — Official crypto symbols
